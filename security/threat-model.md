# PrivateVault Threat Model

## Assets
- Signing Keys
- Audit Receipts
- Policy Definitions
- Replay Logs

## Threats
- Key Compromise
- Log Tampering
- Unauthorized Access
- Replay Attacks
- Supply Chain Attacks

## Mitigations
- Hardware-backed key storage (KMS/HSM)
- Immutable receipt chains
- Namespace isolation
- Fail-closed verification
- Signed container images

## Residual Risk
All residual risks are monitored and reviewed quarterly.
